# 마크다운 문법 정리

## Heading

```
# H1
## H2
### H3
```

## Bold

```
**bold text**
```

ex) **bold text**

## Italic

```
 *italicized text*
```

ex) _italicized text_

## Blockquote

```
> blockquote
```

> ex)
> blockquote

## LIST

### Ordered List

```
1. First item
2. Second item
3. Third item
```

### Unordered List

```
- First item
- Second item
- Third item
```

## Code

```
`code`
```

ex) This is a `code`.

## Horizontal Rule

```
---
```

ex)

paragraph1

---

paragraph2

---

## Link

```
[title](https://www.example.com)
```

ex) [구글](https://google.com)

## Image

```
![alt text](image.jpg)
```
